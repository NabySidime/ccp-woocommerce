# Chap Chap Pay for WooCommerce

Plugin de paiement Chap Chap Pay pour WooCommerce.

## Installation

1. Téléchargez le plugin
2. Upload dans `/wp-content/plugins/`
3. Activez le plugin
4. Configurez dans WooCommerce → Paramètres → Paiements → Chap Chap Pay

## Configuration

- API Key : Votre clé API ChapChapPay
- Clé de chiffrement : Votre clé secrète HMAC
